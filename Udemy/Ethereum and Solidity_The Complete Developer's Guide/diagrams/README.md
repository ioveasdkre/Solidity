圖

課程中顯示的圖表作為zip檔附在本講義中。

下載檔並將其解壓縮到計算機上的某個位置。

參觀 diagrams.net（原 draw.io）。

選擇「打開現有關係圖」 然後使用檔案資源管理員從電腦中選擇關係圖檔。

或

點擊 檔案 從 diagrams.net 功能表中。

選擇「從設備打開」，然後使用文件資源管理員從計算機中選擇關係圖檔。

注意 - 請理解，如果缺少一個圖表或一組圖表，這意味著我們不再有它們可以共用。您需要使用一個好的螢幕截圖瀏覽器擴展程式來複製視頻講座。
